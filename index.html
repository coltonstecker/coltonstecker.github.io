All Star Events

Find events using a city name!

This app allows users to enter a city name and select a corresponding state. From this selection, the app uses an EventBrite API to find events in the searched area. The search also pulls data from the National Parks API to find nearby national parks.

CarCar

The premiere solution for automobile dealership management!

CarCar allows users to build out a system that keeps track of service and sales of vehicles. It retains information such as the customer, salesperson, and technician. This information can be filtered to find specifics, such as all sales made by a specific salesperson or all work done by a technician.

Record Keeper

Keep track of important statistics from trips!

Record keeper allows users to keep track of any information from trips or general experiences. This information can then be filtered down to show experiences from multiple users on the same events. With this information, users can keep track of what went well and what didn't in order to improve future trips.
